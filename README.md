# AI Poem Generator

A React-based web application that generates original poems using
Google's Gemini API. Users can choose from themed categories---Love,
Nature, The Future, and Sorrow---and instantly receive a fresh poem. A
refresh button allows regeneration of new poems within the same theme.

## Features

-   Multiple poem themes: Love, Nature, The Future, Sorrow
-   Refresh button for new poems in the same theme
-   Styled side-by-side layout
-   Real-time poem generation using Gemini API

## Tech Stack

-   React
-   Google Generative AI (@google/genai)
-   CSS

## Installation

    npm install
    npm run dev
